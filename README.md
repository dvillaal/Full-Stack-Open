# Full-Stack-Open — Soluciones y ejercicios

**Repositorio de soluciones a los ejercicios del curso [Full Stack Open](https://fullstackopen.com/).**  

---

## 📚 Descripción

Este repositorio contiene mis soluciones y experimentos para las distintas partes del curso **Full Stack Open**, impartido por la Universidad de Helsinki y colaborantes. Mi objetivo es profundizar mis conocimientos en desarrollo web moderno, tanto frontend como backend, así como prácticas relacionadas (testing, APIs, bases de datos, despliegue, etc.).

Cada “parte” del curso tiene su propio directorio con los ejercicios correspondientes, explicaciones y código funcional.

---

## 🛠 Tecnologías usadas

Algunas de las tecnologías y herramientas que verás presentes:

- **Frontend**: React, JSS, CSS, hooks  
- **Backend / APIs**: Node.js, Express  
- **Bases de datos**: MongoDB  
- **Testing**: Jest, supertest  
- **Git / GitHub** para control de versiones  

---

## ✅ Qué aprendí y retos enfrentados

- Comprensión más profunda del flujo frontend ↔ backend
- Pruebas unitarias y de integración en backend y frontend
- Manejo de estado avanzado, redux, react query
- Buenas prácticas de diseño de API y arquitectura
